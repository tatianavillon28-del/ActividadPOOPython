# Actividad POO en Python

## Descripción
Este programa fue desarrollado aplicando Programación Orientada a Objetos en Python.

Se utilizaron los conceptos de:
- Encapsulamiento
- Herencia
- Composición
- Métodos de comportamiento
- Sobrescritura de __str__()

## Clases desarrolladas
- Vehiculo
- Automovil
- Motocicleta
- Motor

## Funcionamiento
El programa crea automóviles y motocicletas con motores asignados y ejecuta varios métodos de comportamiento.
